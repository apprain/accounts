Method 1: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Click on (Application) tab
    4.	Click on (Install New Component)
    5.	Select accounts.zip file ( Click on the Floppy Disk image on right side of the upload file input box)
    6.	Click on Install Button. 
    7.	After upload, this Accounts component will appear in the list.
        Now click on "Activate" link blow description. 
        
Method 2: 
    1.	First Login to admin panel 
    2.	Go to : (Preference) > (Configuration) 
        Set, Debug Mode = 1
    3.	Unzip accounts.zip.
    4.	Copy "accounts" folder in “component” directory in your project
    5.	In Admin panel go to application tab and find new component
    6.	Click on Activate Link. 
    
 Install Sample data:
    We suggest you install sample data to understand the functionality easily.
     
    1. Go Preference > Database Expert
    2. Go to Import Databse Operation 
    3. Select "sample_data.sql"
    4. Click on "Import" button
    
    
Note: Install another component 'appReport' to view accounting report easily.
